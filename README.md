

VocabTrainer is a simple Android application for learning and practicing English words.

## Features

* Learn new English words
* Practice with simple exercises
* Save words for later review
* Clean and easy-to-use interface

## Technologies

* Kotlin
* Android Studio
* Room Database
* Retrofit (for API if used)

## How to Use

1. Open the app
2. Add or learn new words
3. Practice regularly
4. Track your progress

## Purpose

This project was created to help users improve their English vocabulary in a simple and effective way.

## Future Improvements

* Add more practice modes
* Add AI-based suggestions
* Improve UI/UX
